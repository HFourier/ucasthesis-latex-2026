
# 国科大学位论文模板（2026年）

本模板以《中国科学院大学研究生学位论文撰写规范指导意见（校发学位字〔2022〕40号）》为依据，在国科大Sep系统下载的Latex模板基础上，结合提供的Word模板和指导意见要求，做出的一版符合标准格式的模板，改动内容包括但不限于以下：

- 补充各类参考文献格式，包括会议、在线文献、专利、图书、学位论文等
- 修改参考文献格式为顺序阿拉伯数字右上角引用，取消蓝色高亮
- 修复英文摘要页眉问题
- 正文部分章节不再强制以奇数页开始
- 图目录和表目录和word模板保持一致
- 补充了常用的多图、长表规范性写法
- 补充了单位、变量和公式的规范性要求

# 建议
- 在使用bib参考文献时，几乎每一篇文献都需要重新按照字段进行相应的修改，在使用时，请尽可能不要缺失模板中的字段（部分期刊可能没有卷号），一些期刊网站导出时会有很多多余字段，需要手动删除
- 图片尽可能使用pdf格式图片，这样可以减少整体的pdf大小
- 更多内容和问答，可以参考国科大Latex模板[GitHub](https://github.com/mohuangrui/ucasthesis)
- 如果附录不想加计数，仅显示“附录”，在`ucasthesis.cls`中，Line 661- Line 669，改成如下：
```
\ctexset {
    appendix = {
        numbering = true,
        % number = \rmfamily\zhnum{chapter},
        number = {},
    }
}
```

# 加速编译
随着论文内容增加，latex编译速度会非常慢，因此可以选择仅编译当前章节内容，以加快编译速度，方法如下

1. `Thesis.tex` 文件中，注释掉 Line 93，取消注释 Line 95 - Line 100
2. 取消注释`Thesis.tex` 文件中的Line 63，里面改成自己当前撰写的章节tex文件就行

本文模板在满足《中国科学院大学研究生学位论文撰写规范指导意见（校发学位字〔2022〕40号）》的依据下，参考了多篇中国科学院百篇优秀博士论文格式，进行了相关整理，各院所可能存在差异，使用模板时仍需仔细对比和校对。
